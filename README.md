# Teste de Integração em Java

Este projeto demonstra um teste de integração simples utilizando Java.

## 📌 Objetivo
Simular um sistema de login e validar o funcionamento em três cenários:
- Login com erro
- Login correto
- Código melhorado

## 🧪 Testes realizados

### ❌ Versão com erro
O sistema recebe dados inválidos e falha na autenticação.

### ✅ Versão correta
O sistema autentica corretamente com usuário e senha válidos.

### ⭐ Versão melhorada
O código foi aprimorado com melhor tratamento de erros e organização.

## ▶️ Como executar

1. Abra no NetBeans
2. Execute a classe `Main`
3. Observe os resultados no console
